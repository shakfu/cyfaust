# cyfaust

A cython c++ wrapper for the faust interpreter.


## package design


```
cyfaust
	__init__.py
	cyfaust.so
	faust_interp.pxd
	faust_box.pxd
	faust_box.pxi
	faust_signal.pxd
	faust_signal.pxi
```
